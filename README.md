#Descripción 
La media tecnica de Stfware es una modalidad o programa educativo que e prepara para ser bueno en el desarrollo y formación que combina la teoria y la práctica para que aprendas a diseñar,desarrollar y mantener sotfware de alta calidad,durante este programa estudiamos temas como:
programación en diferenes lenguajes "Java,Python,c++,etc.".
Desarrollo de aplicaciones web y moviles
Bases de datos y gestión de datos
Sguridad informática
Análisis y diseño de sistemas
Con esta media tecnican podemos trabajar es en empresas de tecnologia,desarrollar proyectos propios o despues de terminar la media tecnica seguir estudiando en áreas como la ingenieria de stfware o la inteligencia artificial.
